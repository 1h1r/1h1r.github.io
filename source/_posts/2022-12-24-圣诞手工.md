## 圣诞手工DIY🎄

2022年圣诞节前夕，亦然亦航决定亲手制作圣诞礼物送给家人和好朋友。

### 准备工作

**材料清单：**
- 彩色卡纸（红、绿、金、白）
- 剪刀、胶水、双面胶
- 彩色丝带
- 毛线和钩针
- 小铃铛
- 烘焙材料（做姜饼人）
- 空白贺卡和彩色笔

### 亦然的作品：3D立体圣诞贺卡

亦然决定用卡纸制作一张立体的圣诞贺卡。

**制作步骤：**

1. **设计图纸**
   - 用铅笔在卡纸上画出教堂、圣诞树和雪人的轮廓
   - 设计弹出式立体结构

2. **裁剪零件**
   - 教堂主体（5个部件）
   - 圣诞树（3层）
   - 雪人和雪花（10个）

3. **组装结构**
   ```
   折叠方式：
   - V型折叠：教堂的屋顶
   - 平行折叠：圣诞树的层次
   - 旋转结构：雪人的头部
   ```

4. **上色装饰**
   - 用彩色笔给教堂上色
   - 金色丝带做圣诞树装饰
   - 小铃铛挂在贺卡封面

5. **写祝福语**
   > "亲爱的爸爸妈妈，圣诞快乐！感谢你们的爱和陪伴。爱你们的亦然"

**制作时间：** 3小时
**成品效果：** 打开贺卡时，教堂、圣诞树和雪人立体弹出，非常精美！

### 亦航的作品：毛线钩织圣诞袜

亦航虽然手不太巧，但他决定挑战钩织一个圣诞袜。

**学习过程：**

亦航跟着YouTube教程学了一个下午，终于掌握了基本的钩针技巧：

- 锁针起针
- 短针
- 引拔针
- 换线

**制作过程：**

```
圣诞袜钩织图案：
第1行：锁针20针
第2-10行：短针20针（袜口部分，红白条纹）
第11-15行：短针加针（袜身部分，绿色）
第16-25行：短针减针（袜尖部分，红色）
最后：缝上白色毛边和挂绳
```

虽然亦航的作品有些歪歪扭扭，但他说：
> "虽然不太好看，但这是我一个针一个针钩出来的！里面有我的爱！"

**制作时间：** 5小时（拆了3次重做）
**成品效果：** 一个歪歪扭扭但充满爱的圣诞袜！

### 一起做姜饼人

最有趣的是全家一起做姜饼人！

**姜饼人配方：**
```
材料：
- 低筋面粉 300g
- 黄油 100g
- 红糖 80g
- 蜂蜜 100g
- 鸡蛋 1个
- 姜粉 1小勺
- 肉桂粉 1小勺
- 泡打粉 1小勺

步骤：
1. 黄油软化，加红糖打发
2. 加入蜂蜜和鸡蛋搅拌均匀
3. 筛入面粉和调料，揉成面团
4. 冷藏1小时
5. 擀平，用模具切出形状
6. 180°C烤15分钟
7. 用糖霜装饰
```

**分工合作：**

| 任务 | 负责人 | 完成情况 |
|------|--------|---------|
| 称量材料 | 亦然 | 精确到克 ✅ |
| 揉面团 | 爸爸 | 均匀光滑 ✅ |
| 擀面 | 亦航 | 厚薄不一 😅 |
| 切模具 | 亦然 | 形状完美 ✅ |
| 烤制 | 爸爸 | 金黄酥脆 ✅ |
| 装饰 | 亦然+亦航 | 创意十足 ✅ |

**姜饼人装饰创意：**

亦然做的姜饼人：
- 穿着"实验服"的科学家姜饼人
- 戴眼镜的"学霸"姜饼人
- 拿着烧杯的"化学家"姜饼人

亦航做的姜饼人：
- 穿球衣的"足球小将"姜饼人
- 踢球姿势的"运动员"姜饼人
- 戴奖牌的"冠军"姜饼人

### 礼物交换

圣诞夜，全家围在圣诞树旁交换礼物：

1. 亦然送给妈妈的立体贺卡 → 妈妈感动落泪
2. 亦航送给爸爸的歪歪扭扭圣诞袜 → 爸爸笑得合不拢嘴
3. 姜饼人互相赠送 → 亦然拿了亦航的足球姜饼人，亦航拿了亦然的科学家姜饼人
4. 爸爸妈妈送的礼物：亦然得到了一套科学实验套装，亦航得到了一个新足球

### 成长收获

这次圣诞手工活动带来的收获：

**亦然：**
- 锻炼了精细动手能力

- 体验了从设计到成品的全过程

- 学会了新技能（钩针编织）

- 体会到了为他人付出的快乐

> 最好的礼物不是最贵的，而是最用心的。每一针、每一刀、每一笔，都是爱的表达。❤️
