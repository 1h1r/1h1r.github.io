# 亦然亦航 - 双胞胎成长博客

> 用爱记录双胞胎男孩亦然与亦航从幼儿园到大学的成长历程

## 关于博客

这是亦然亦航双胞胎兄弟的成长博客，记录了他们从2020年上幼儿园到2039年大学毕业的完整成长历程。

博客内容涵盖：
- **成长日记** - 每个成长阶段的趣事和里程碑
- **兴趣爱好** - 科学探索、体育运动、音乐艺术、编程机器人
- **趣味生活** - 家庭旅行、节日庆祝、手工DIY
- **学校课程** - 各阶段的学习笔记和校园生活

## 技术栈

- **博客框架：** [Hexo](https://hexo.io/)
- **主题：** [Butterfly](https://github.com/jerryc127/hexo-theme-butterfly) v5.5.5
- **部署：** GitHub Pages
- **域名：** https://1h1r.github.io

## 本地运行

```bash
# 安装依赖
npm install

# 本地预览
hexo server

# 生成静态文件
hexo generate
```

## 博客结构

```
├── _config.yml              # Hexo 配置
├── _config.butterfly.yml    # Butterfly 主题配置
├── source/
│   ├── _posts/              # 博客文章
│   ├── about/               # 关于页面
│   ├── categories/          # 分类页面
│   ├── tags/                # 标签页面
│   ├── gallery/              # 相册页面
│   ├── css/custom.css       # 自定义样式
│   └── js/custom.js         # 自定义脚本
├── .github/workflows/       # GitHub Actions 自动部署
└── package.json
```
